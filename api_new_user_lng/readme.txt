New Linnaeus user API

Usage:
http://url/plugins/api_new_user_lng/  [needs POST variables as set below]

Parameters:
username=[string]            Linnaeus user name

The Linnaeus user name is used to create a new ResourceSpace user with this name.
A "My collection" private collection is created automatically.
The service returns the ResourceSpace credentials to log in and
to use the search and upload api's.
